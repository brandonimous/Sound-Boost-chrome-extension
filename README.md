# Sound Booster 900%

Amplificador de audio para Chrome.

**Version:** 0.0.1

## Instalacion

1. Abre `chrome://extensions/` en Chrome.
2. Activa **Modo desarrollador**.
3. Haz clic en **Cargar descomprimida** y selecciona la carpeta del proyecto (la que contiene `manifest.json`).

## Uso

- Abre una pestaña con audio (YouTube, Spotify Web, etc.).
- Haz clic en el icono de la extension y ajusta el volumen desde el popup.

## Desarrollo rapido

- Edita `popup.html` / `popup.js` / `content.js` y recarga la extension en `chrome://extensions/`.
